# Beam Stress Distribution Analysis using ANSYS

Performed Finite Element Analysis (FEA) on a structural beam to evaluate stress distribution and identify critical stress concentration regions under applied loading conditions.

## 🚀 Project Overview
This project focused on understanding structural behavior using simulation-driven engineering methods through the complete FEA workflow in ANSYS.

## 🛠️ Tools & Technologies
- ANSYS Student Edition
- Finite Element Analysis (FEA)
- Structural Simulation

## ⚙️ Analysis Workflow
- Geometry preparation
- Material assignment
- Meshing
- Boundary condition setup
- Load application
- Solver execution
- Result interpretation

## 📌 Key Objectives
- Evaluate von Mises stress distribution
- Identify critical stress concentration zones
- Validate structural integrity under loading
- Understand deformation behavior

## 📊 Simulation Results
- Observed stress concentration near constrained regions
- Evaluated structural response under applied load
- Verified design safety against failure thresholds

## 📈 Outcomes
- Gained hands-on experience in simulation-driven engineering
- Improved understanding of structural analysis principles
- Demonstrated practical application of FEA methodologies

ANSYS • FEA • Structural Analysis • Meshing • Stress Analysis • Engineering Simulation • Mechanical Design Validation
